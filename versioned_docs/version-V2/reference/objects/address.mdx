The `Address` type provides an address information on multiple EVM chains.

```graphql
type Address {
  address: AddressEVM!
  ethWallet: Wallet
  goerliWallet: Wallet
  identity: Identity!
  followers(
  offset: Int
  limit: Int
): FollowConnection!
  followings(
  offset: Int
  limit: Int
): FollowConnection!
  followStats: FollowStats!
  isFollowedBy(
  addresses: [AddressEVM!]!
): [Boolean!]!
  isFollowing(
  addresses: [AddressEVM!]!
): [Boolean!]!
}
```

#### `goerliWallet` ([`Wallet`](/reference/objects/wallet))
