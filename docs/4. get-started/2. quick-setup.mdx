### Prerequsites
Make sure MetaMask is installed


### CyberConnect SDK:

We'll be working with the CyberConnect SDK:

```bash npm2yarn
npm install @cyberlab/cyberconnect
```


### Queries

For queries you'll need Install both packages:

```bash npm2yarn
npm install graphql graphql-request
```
