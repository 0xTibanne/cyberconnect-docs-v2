`address` returns information about an EVM address.

```graphql
address(
  address: AddressEVM!
): Address
```


### Arguments

#### `address` ([`AddressEVM`](/reference/scalars/address-evm))




### Type

#### [`Address`](/reference/objects/address) 

The `Address` type provides an address information on multiple EVM chains.
