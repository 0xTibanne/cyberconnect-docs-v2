The `CreateSetEssenceDataTypedDataInput` input type specifies the params required to set the essence data.

```graphql
input CreateSetEssenceDataTypedDataInput {
  options: TypedDataOptions
  profileId: Int!
  essenceId: Int!
  tokenURI: URL!
  mwAddress: ContractAddressEVM!
}
```


### Fields

#### `options` ([`TypedDataOptions`](/reference/inputs/typed-data-options))

`options` the options to set the essence data.


#### `profileId` ([`Int`](/reference/scalars/int))

`profileId` the id of the profile that is setting the essence data.


#### `essenceId` ([`Int`](/reference/scalars/int))

`essenceId` the id of the essence that is having its data set.


#### `tokenURI` ([`URL`](/reference/scalars/url))

`tokenURI` the token URI of the essence.


#### `mwAddress` ([`ContractAddressEVM`](/reference/scalars/contract-address-evm))

`mwAddress` the address of the middleware contract.
