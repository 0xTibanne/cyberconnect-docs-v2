The `CreateSetSubscribeDataTypedDataInput` input type specifies the params required to set a subscribe middleware.

```graphql
input CreateSetSubscribeDataTypedDataInput {
  options: TypedDataOptions
  profileId: Int!
  tokenURI: URL!
  mwAddress: ContractAddressEVM!
}
```


### Fields

#### `options` ([`TypedDataOptions`](/reference/inputs/typed-data-options))

`options` the options to set a subscribe middleware.


#### `profileId` ([`Int`](/reference/scalars/int))

`profileId` the ID od the profile setting the subscribe middleware.


#### `tokenURI` ([`URL`](/reference/scalars/url))

`tokenURI` the token URI of the subscribe NFT.


#### `mwAddress` ([`ContractAddressEVM`](/reference/scalars/contract-address-evm))

`mwAddress` the contract address of the subscribe middleware.
